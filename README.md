## Overview
Drum Machine is an interactive browser-based application that allows users to play drum sounds using keyboard keys or mouse clicks. The project simulates a basic beat-making experience and focuses on event handling and audio playback in JavaScript.

---

## Features
- Real-time drum sound playback
- Keyboard and mouse input support
- Simple and intuitive UI
- Low-latency audio triggering
- Lightweight and fast

---

## Tech Stack
- HTML
- CSS
- JavaScript
- Browser Audio API
